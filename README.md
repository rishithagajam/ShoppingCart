# Shopping Cart 🛒
A web-based Shopping Cart application built with **HTML**, **CSS**, and **JavaScript** that allows users to browse products, add them to a cart, and manage their purchases with a smooth and responsive interface.

## 📖 Project Overview
This project is designed to provide a seamless online shopping experience. With a clean and responsive UI, users can browse through products, search for items, and manage their cart efficiently. The application is built entirely with vanilla web technologies — no frameworks required.

---

## ✨ Key Features
- 🛍️ **Add & Remove Items**: Easily add or remove products from the cart
- 🔍 **Search Functionality**: Quickly find products using the search bar
- 💰 **Dynamic Price Calculation**: Cart total updates in real-time
- 📱 **Responsive Design**: Works smoothly on all screen sizes
- ⚡ **Fast & Lightweight**: No frameworks or dependencies needed
- 🎨 **Clean Modern UI**: Simple and intuitive user interface

---

## 🛠️ Technologies Used
- **HTML5** - Structure and layout of the web pages
- **CSS3** - Styling, animations, and responsive design
- **JavaScript (Vanilla)** - Cart logic and interactivity
- **Version Control**: Git & GitHub

## 📥 Installation

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge)
- Git

### Steps

1. **Clone the repository**
```bash
git clone https://github.com/rishithagajam/ShoppingCart.git
cd ShoppingCart
```

2. **Open the project**
```bash
# Simply open index.html in your browser
start index.html       # Windows
open index.html        # macOS
xdg-open index.html    # Linux
```

> No server setup or installation required!

---

## 💻 Usage
1. Open `index.html` in your browser
2. Browse through the available products
3. Click **Add to Cart** to add items
4. Use the **Search bar** to find specific products
5. View your cart and see the total price update dynamically
6. Remove items from the cart as needed
